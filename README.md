# robosys2022
このレポジトリは千葉工業大学未来ロボティクス学科のロボットシステム学の講義で作成した。

# plus

* 機能

入力した数字以下の自然数を足して答えを表示する。

* 使い方

```
seq [自然数] | ./plus
```

# 動作確認済み環境

* テスト済み　Python 3.7～3.10

# インストール方法

* 以下のコマンドを実行

```
git clone https://github.com/TakumiIshiguro/robosys2022.git
```
